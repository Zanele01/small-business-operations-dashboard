# Small Business Operations Dashboard

A full-stack offline-first business management application designed for small businesses.

## Features

- Inventory Management
- Sales Tracking
- Expense Management
- Analytics Dashboard
- Offline-First Support
- Progressive Web App

## Tech Stack

Frontend:
- React
- Vite

Backend:
- Node.js
- Express

Storage:
- JSON Persistence

Version Control:
- Git
- GitHub

Deployment:
- Vercel
- Render
